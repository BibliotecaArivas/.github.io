Guía para Instalar Python en Windows

Esta guía te mostrará paso a paso cómo instalar Python en tu computadora con Windows, incluyendo cómo configurarlo correctamente.

Paso 1: Descargar Python

  1. Abre tu navegador web y ve al sitio oficial de Python
  2. Haz clic en la pestaña "Downloads" (Descargas)
  3. El sitio detectará automáticamente tu sistema operativo y te mostrará un botón para descargar la última versión de Python para Windows
Descargar Python

Alternativa: Si necesitas una versión específica, desplázate hacia abajo y selecciona "Windows" en la sección "Looking for a specific release?"

Paso 2: Ejecutar el instalador

  1. Abre el archivo descargado (ej. python-3.x.x.exe)
  2. Marca la casilla "Add Python to PATH" (ES IMPORTANTE para poder usar Python desde cualquier lugar)
  3. Haz clic en "Install Now" (Instalar ahora)

Opciones avanzadas: Puedes hacer clic en "Customize installation" si necesitas:

  • Instalar para todos los usuarios
  • Cambiar la ruta de instalación
  • Instalar características adicionales

Paso 3: Completar la instalación

  1. Espera a que termine el proceso de instalación (puede tomar unos minutos)
  2. Cuando termine, verás un mensaje "Setup was successful"
  3. Haz clic en "Close" (Cerrar)

Paso 4: Verificar la instalación

  1. Abre el Símbolo del sistema (CMD):
    • Presiona Win + R, escribe cmd y presiona Enter
  2. En la ventana que aparece, escribe: 
    python --version
  3. Deberías ver la versión de Python instalada (ej. "Python 3.x.x")

También puedes probar:

pip --version

Para verificar que el gestor de paquetes está instalado

Paso 5: (Opcional) Instalar un IDE/Editor

Para programar en Python te recomiendo instalar:

  • VS Code: https://code.visualstudio.com/
    (Luego instala la extensión de Python)
  • PyCharm Community: https://www.jetbrains.com/pycharm/
  • Thonny (ideal para principiantes): https://thonny.org/

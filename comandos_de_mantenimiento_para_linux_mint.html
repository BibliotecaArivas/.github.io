Comandos de Mantenimiento

Linux Mint - Guía completa para mantener tu sistema optimizado

Esta página contiene una colección completa de comandos útiles para el mantenimiento de tu sistema Linux Mint. Haz clic en cualquier comando para copiarlo al portapapeles.

Nota: Algunos comandos requieren privilegios de administrador (sudo).

Mantenimiento del Sistema

Actualizar lista de paquetes
sudo apt update
Actualiza la lista de paquetes disponibles desde los repositorios.
Actualizar paquetes instalados
sudo apt upgrade
Actualiza todos los paquetes instalados a sus versiones más recientes.
Actualización completa
sudo apt full-upgrade
Actualización completa que puede eliminar paquetes obsoletos si es necesario.
Limpiar paquetes innecesarios
sudo apt autoremove
Elimina paquetes que ya no son necesarios.
Limpiar caché de paquetes
sudo apt autoclean
Elimina archivos de paquetes antiguos de la caché.

Gestión de Paquetes

Buscar paquetes
apt search [nombre]
Busca paquetes en los repositorios.
Información de un paquete
apt show [paquete]
Muestra información detallada sobre un paquete.
Listar paquetes instalados
apt list --installed
Muestra todos los paquetes instalados en el sistema.
Reparar dependencias
sudo apt --fix-broken install
Intenta reparar dependencias rotas de paquetes.
Reparar configuración de paquetes
sudo dpkg --configure -a
Reconfigura paquetes que no se instalaron correctamente.

Mantenimiento de Disco

Ver espacio en disco
df -h
Muestra el espacio usado y disponible en los discos.
Ver tamaño de directorios
du -sh /directorio/*
Muestra el tamaño de los directorios de forma legible.
Encontrar archivos grandes
find / -type f -size +100M 2>/dev/null
Encuentra archivos de más de 100MB en el sistema.
Limpiar caché de paquetes
sudo apt clean
Elimina todos los archivos de paquetes de la caché.
Verificar disco
sudo fsck /dev/sda1
Verifica y repara el sistema de archivos (ejecutar sin montar).

Monitoreo del Sistema

Ver procesos en ejecución
top
Muestra los procesos en ejecución y el uso de recursos.
Monitor interactivo
htop
Versión mejorada de top con interfaz más amigable.
Uso de memoria
free -h
Muestra el uso de memoria RAM y swap.
Información del sistema
hostnamectl
Muestra información sobre el sistema y el host.
Ver logs del sistema
journalctl -f
Muestra los logs del sistema en tiempo real.

Limpieza del Sistema

Limpiar archivos temporales
sudo rm -rf /tmp/*
Elimina todos los archivos del directorio temporal.
Limpiar caché del usuario
rm -rf ~/.cache/*
Elimina la caché del usuario actual.
Limpiar logs antiguos
sudo journalctl --vacuum-time=7d
Elimina logs del sistema más antiguos de 7 días.
Limpiar kernels antiguos
sudo apt autoremove --purge
Elimina kernels antiguos y sus archivos de configuración.
Limpiar thumbnails
rm -rf ~/.cache/thumbnails/*
Elimina las miniaturas de archivos en caché.

Comandos de Emergencia

Recuperar GRUB
sudo update-grub
Actualiza la configuración del gestor de arranque GRUB.
Instalar GRUB
sudo grub-install /dev/sda
Reinstala GRUB en el disco principal.
Forzar cierre de proceso
sudo kill -9 [PID]
Termina forzosamente un proceso por su ID.
Matar proceso por nombre
sudo pkill [nombre]
Termina todos los procesos con un nombre específico.
Script de mantenimiento rápido
sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y && sudo apt autoclean
Ejecuta varias operaciones de mantenimiento en una sola línea.

Consejos de Mantenimiento Regular

Diariamente

Ejecuta sudo apt update para mantener actualizada la lista de paquetes.

Semanalmente

Ejecuta sudo apt upgrade && sudo apt autoremove para mantener el sistema actualizado.

Mensualmente

Realiza una limpieza de logs y caché para liberar espacio en disco.

Trimestralmente

Verifica la salud de tus discos y realiza backups de tus datos importantes.

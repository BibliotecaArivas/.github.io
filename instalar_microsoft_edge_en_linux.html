Instalar Microsoft Edge en Linux

Instalación mediante archivo DEB o RPM

  • Ve al sitio oficial de Microsoft Edge.
  • Haz clic en Descargar para Linux y elige el archivo .deb (Debian/Ubuntu) o .rpm (Fedora/RHEL).
  • Acepta los términos y descarga el paquete.
  • Instálalo con tu gestor de software (como Software Center) o desde terminal:
    • Debian/Ubuntu: sudo dpkg -i nombre-del-archivo.deb
    • Fedora/RHEL: sudo dnf install nombre-del-archivo.rpm
  • Abre Edge desde el menú de aplicaciones o ejecuta microsoft-edge en la terminal.

Instalación mediante repositorio (Ubuntu/Debian)

Este método permite recibir actualizaciones automáticas a través de apt.

  • Descarga e instala la clave GPG:
curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
  • Añade la clave al sistema:
sudo install -o root -g root -m 644 microsoft.gpg /etc/apt/trusted.gpg.d/
  • Agrega el repositorio:
echo "deb [arch=amd64] https://packages.microsoft.com/repos/edge stable main" | sudo tee /etc/apt/sources.list.d/microsoft-edge-stable.list
  • Limpia el archivo temporal y actualiza:
sudo rm microsoft.gpg && sudo apt update
  • Instala Microsoft Edge:
sudo apt install microsoft-edge-stable
  • Inicia el navegador:
microsoft-edge

Notas adicionales

  • El repositorio se agrega automáticamente durante la instalación vía terminal, lo que garantiza actualizaciones futuras.
  • Para desinstalar completamente: 
    sudo apt remove --purge microsoft-edge-stable && sudo rm /etc/apt/sources.list.d/microsoft-edge-stable.list
  • El contenido del repositorio está disponible en packages.microsoft.com/repos/edge.

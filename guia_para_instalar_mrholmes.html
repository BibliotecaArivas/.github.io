Guía para Instalar MrHolmes en Kali Linux

Herramienta avanzada de OSINT para recopilación de información de fuentes públicas

Repositorio Oficial

Instalación de MrHolmes

Requisitos Previos

sudo apt update && sudo apt upgrade -y
sudo apt install git python3-pip -y

Proceso de Instalación

1. Clonar el repositorio:

git clone https://github.com/LimerBoy/MrHolmes.git
cd MrHolmes

2. Instalar dependencias:

pip3 install -r requirements.txt

3. Configurar claves de API (opcional):

nano config/api_keys.json

Agrega tus claves de servicios como Shodan, Hunter.io, etc.

Uso de MrHolmes

python3 mrholmes.py
# O con permisos de ejecución:
chmod +x mrholmes.py
./mrholmes.py

Comandos Principales

Comando Descripción
--email example@domain.com Busca información relacionada con un correo electrónico
--username john_doe Busca un nombre de usuario en redes sociales
--domain example.com Realiza búsquedas OSINT sobre un dominio
--phone +123456789 Busca información asociada a un número telefónico

Ejemplos de Uso

1. Búsqueda por Correo Electrónico

./mrholmes.py --email target@example.com

Busca fugas de datos, redes sociales y más asociadas al correo.

2. Búsqueda por Nombre de Usuario

./mrholmes.py --username hacker123

Verifica la presencia del usuario en plataformas como GitHub, Twitter, Instagram, etc.

3. Análisis de Dominio

./mrholmes.py --domain example.com

Extrae subdominios, registros WHOIS y posibles vulnerabilidades.

Solución de Problemas

1. Error de Dependencias Faltantes

pip3 install --force-reinstall -r requirements.txt

2. Problemas con APIs Externas

Asegúrate de tener claves válidas en config/api_keys.json.

3. Permisos Denegados

chmod +x mrholmes.py
./mrholmes.py
